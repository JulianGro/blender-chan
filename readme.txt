※豆知識もとい黒歴史：もともとKritaはLinuxのKDE環境用に作られたGIMPのフォーク(派生ソフト)で、当初の名前はKIMPだった。初めは色々な機能をぶっこんだKDE用のGIMPといった感じだったが、結構重く、しかもバグが多く、ぶっちゃけ本家GIMPの方がより安定してをり、軽く、かつ高性能だった（「KritaはGIMPに劣る」偏見はこれが元凶）。そこでKrita(Krayonと呼ばれてた)は、GIMPやPhot○shopといったいわゆる「画像編集ソフト」になるのをあきらめ、完全に別のルートへ進む事を決意。後に「つよつよ画像"作成"ソフト」として名が知られるようになり今日へ至る。現在Kritaは、他の画像ソフトとは全く異なるかなりクセの強い操作性で油繪や水彩畫等をゼロから描くことを得意とした最強のオープンソースペインティングソフトウェアだ。#キキちゃんかわいいよｈｓｈｓ #君もKritanにならないか？

＃【ブレンダーちゃんをVRChatへアップロードしようぜ手順】＃
＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃
①VRChatで使はれている最新のUnityを https://docs.vrchat.com/docs/current-unity-version から確かめよう！このreadmeを書いた時は　Unity2019.4.29f1 でした。

②上記のUnity(2019.4.29f1)をインストールして、新しいプロジェクトを開きませう！

③最新のVRCSDK3-Avatars（コレを作ったときはVRCSDK3-AVATAR-2021.08.04.11.23_Publicだったべ）を https://vrchat.com/home/download からダウンロードして、Unityプロジェクト内のAssetsへドラッグ＆ドロップ＆インポートしませう！

④（お好みで）Lyuma氏の神AV3エミュレーターを https://github.com/lyuma/Av3Emulator から導入しましょう。これを使ふと一一VRChatへアップロードしなくてもUnity上からVRChatでの擧動を大体把握することが出来ます。ありがとうLyumaさんっ！！

⑤本ファイルに同梱されてゐる Blender-chan_v1.08f.unitypackage をプロジェクトのAssetsへドラッグ＆ドロップでインポートしませう。

⑥無事インポートに成功すれば、Assetsフォルダ内に anim, ExpressionsMenu, Icons, Locomotion, Materials, Scenes, UmebocDC, v1.08, (VRCSDK), というフォルダーと、 neko_3 というユニティシーンがどっかに出てくるハズです。この最後の neko_3 と書かれてゐるシーンをなんとかして開いてください。

⑦シーンを開くと今度はヒエラルキーにneko_3と言う名前とともに blender-chan_v1.08b という青いパッケージ（通称「ぷりふぁぶ」）が出現するハズです。この青いパッケージをクリックすると、インスペクタータブになんか色々（Pipeline Manager（いらない子）と VRC Avatar Descriptorが）出現するハズです。
・以下「VRC Avatar Descriptorなんてなかった」人向け
　VRC Avatar Descriptorを手動で設定しませう。ヒエラルキーのneko_3シーン内にある青い箱(blender-chan_v1.08b)をクリックし、インスペクタータブから「コンポーネントを追加」します。
　VRC Avatar Descriptor内のView Position を X:0, Y:1.41, Z:0.04 に設定します。
　LipSyncはAuto Detect!で自動的に入力されそうですが、運悪くなんかダメっぽくなった場合は手動でやりませう。
　
　Mode: Viseme Blend Shape
　Face Mesh: body (SkinnedMeshRenderer)
　Viseme: sil: sil
　Viseme: PP: PP
　Viseme: FF: FF
　Viseme: TH TH
　Viseme: DD: DD
　Viseme: kk: KK
　Viseme: CH: CH
　Viseme: SS: SS
　Viseme: nn: NN
　Viseme: RR: RR
　Viseme: aa: a_AA
　Viseme: E: e_E
　Viseme: ih: i_IH
　Viseme: oh: o_OH
　Viseme: ou: u_OU
　
　次にEye Look を Enable! しませう。以下の設定をします。
　Eyes 
　　Transforms
　　　Left Eye Bone: Eye.L
　　　Right Eye Bone: Eye.R
　　Rotation States
　　　Looking Straight
　　　　X:-90 Y:180 Z:0
　　　Looking Up
　　　　X:-75 Y:180 Z:0
　　　Looking Down
　　　　X:-85 Y:0 Z:180
　　　Looking Left
　　　　X:-90 Y:160 Z:0
　　　Looking Right
　　　　X:-90 Y:-160 Z:0
　Eyelids
　　Eyelid Type: Blendshapes
　　Eyelids Mesh: body (SkinnedMeshRenderer)
　　Blendeshape States
　　　Blink: blink
　　　Looking Up: none
　　　Looking Down: none
　
　次にPlayable Layersを設定します。変えるのはBaseとFXだけです。
　Base
　　Base: Blender-chan_Locomotion
　　Additive: Default Idle
　　Gesture: Default Gesture
　　Action: Default Action
　　FX: Default Blender_chan_HandsLayer
　
　最後にExpressionsを設定します。
　Expressions
　　Menu: Blender_chan_ExMainMenu
　　Parameters: Blender_chan_ExPara


⑧UnityのメニューにあるVRCSDKからShow Control Panelをクリックして、VRChatSDKにログインし、なにかエラーがあったら対処します。エラーがなければそのままアップロードしてあげます。

⑨これで終わりです！アバターアップロードおつかれさまです！


【MeにでもわかるQ&A】＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃＃
Q. これはなに？
A. きゅーあんどえいだよ！英語で「質問と回答」（Questions and Answers）を意味するよ！

Q. いやそうじゃなくて、このアバターのこと😅
A. あっリオ式「ブレンダーちゃん」は、世界的に有名な自由とフリーのオープンソース3Dソフトウェア"Blender"の非公式二次創作擬人化キャラクターだよ！

Q. 自作？
A. 自作だよ！でも、元ネタはあるよ! https://sketchfab.com/3d-models/blender-chan-6835f0d60e0c4813812c0247e3b73da7
香港にいるSearKitchenさんって方が2017年2月12日に最初の"Blender-chan!"をSketchfabっていうサイトで公開してて、そのSearKitchenさんの三面図(スケッチ)をもとに独自解釈してゼロから作ったのが本作リオ式「ブレンダーちゃん」なのですー。SearKitchenさんの作った"Blender-chan!"はいい感じの骨(IK)がちゃんと組まれてて、Blenderでのアニメーションに特化してるのに対して、川音さんの作ったリオ式「ブレンダーちゃん」は、VRChatっていうゲームの中で使われることを想定したローポリモデルなのが一番の違いだよ！ #VRChat想定モデル	

Q. Blenderのロゴ勝手に使っちゃっていいの？
A. 質問が曖昧！一応ロゴの規約は公式サイト https://www.blender.org/about/logo/ を参考にしてて、とりあえずウチは別に利益だしてないし、二次創作という形で提供してるのでセーフなハズ。無論、Blender本社からブレンダーちゃんの配布ダメって言われたら配布止めますが...。もしこれがBlenderのロゴを使ひたいユーザーさんからの質問なら、Blenderの公式サイトに目を通して下さいな。

3. Download de nieuwste VRCSDK3-Avatars van https://vrchat.com/home/download (VRCSDK3-AVATAR-2021.08.04.11.23_Public op het moment van schrijven) en sleep het naar Assets in het Unity Project om het te importeren!

4. (Optioneel) Installeer Lyuma's AV3 Emulator van https://github.com/lyuma/Av3Emulator

5. Sleep Blender-chan_v1.08f.unitypackage naar de Assets van het project.

6. Als de import gelukt is, vindt u in de map Assets de volgende mappen: anim, ExpressionsMenu, Icons, Locomotion, Materials, Scenes, UmebocDC, v1.08, (VRCSDK), en neko_3. Deze "neko_3" is een unityscene. Open deze scène.

　Stel de View Position in de VRC Avatar Descriptor in op X:0, Y:1.41, Z:0.04.
